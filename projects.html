Bowhead Coil : Machine learning Marketplace

Devised an online machine learning marketplace in collaboration with Bowhead Health Inc. which provides a platform for all the Researchers and Developers for a secured exchange of models. The project was created using React JS, ES6 JavaScript, Material UI and Jupyter notebook. The code is private since its an intellectual property of the company.

Roots- Android application

Created an android application imparting information about the routes and stops of the local transportation used in Jammu (J&K, India). The guidance app worked by letting the user enter the source and destination and popped the data containing the calculated fare, best possible route with a visual enhancement of Google maps. It used a regional transport database including the location of the stop, vehicle allotted for every route, and price for the distance covered and the mobile backend used was Parse for backend & BaaS for frontend programming
